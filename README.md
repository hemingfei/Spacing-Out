# Spacing-Out
基于Unity的角色扮演类音乐游戏
***
## 项目说明
上传资源包括动画、材质、模型、音频文件、Unity场景以及C#脚本
音游功能主要利用插件Koreographer实现
PS：当前项目资源内不包含插件资源，提供代码和模型素材以供参考
